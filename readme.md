# Passwall -- 让您的代理之路所向披靡 #

## SS,SSR,V2RAY,TROJAN于一体的Passwall软件 ##

## 支持Windows，Mac，Linux全平台 ##

## 使用PyQt5编写，界面友好、美观，适合小白操作 ##

***

### 众所周知，由于中国的法律限制，部分境外网站在国内无法直接连接（或者速度非常慢），而大部分的代理软件仅支持部分代理方式（协议），又或者仅支持部分操作系统，故开发本软件。 ###
